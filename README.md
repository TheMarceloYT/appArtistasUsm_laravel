App creada en laravel 10 y bootstrap 5.3 con bootstrap ICONS a nivel PROFESIONAL, usando XAMPP para la gestion de registros de la base de datos con Apache y MySql

============================================================================================

(Aplicación aún en proceso de mejora, trabajando en refinar diseño front-end y funcionalidades back-end).

App que emula una extensión de la pagina para la USM por parte de los usuarios Artistas, pudiendo subir sus imagenes 
con sus atributos (titulo, descripcion, etc), y ver los comentarios que se le hagan a sus publicaciones y likes,
como usuario Estudiante puedes ver todas las publicaciones y comentarlas para dar tu opinión y darle like si esque
te gustaron, como usuario normal puede ver todas las publicaciones de los artistas, finalmente como usuario administrador
puede gestionar (borrar, editar, ver y añadir) Artistas, administradores, estudiantes y poder eliminar publicaciones no deseadas.

Uso completo de todas las herramientas, tanto de laravel 10 => para iniciar sesión (Auth), seguridad (middleware, gates, etc), 
plantillas (blade), validadores (Request personalizados, rules, etc), relaciones en base de datos de ELOQUENT con usos de llenado
de datos (migrations, seeders, faker, factory), uso a nivel PROFESIONAL.

La app puede proveer de datos de prueba para que pueda ver el funcionamento a fondo del sistema, usando el comando en 
la terminal => php artisan migrate:fresh --seed , podrá llenar su base de datos (accede en el archivo .env usted poniendo sus 
credenciales de la base de datos).
